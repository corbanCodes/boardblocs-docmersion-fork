## 3. Technical SEO Requirements

### 3.1 robots.txt Requirements

**Location:** /robots.txt

**Required Elements:**
- [ ] User-agent directives for all bots
- [ ] Sitemap location specified
- [ ] Crawl-delay set (recommended: 1 second)
- [ ] Disallow rules for non-indexable content
- [ ] Allow rules for important resources
- [ ] Bot-specific throttling for aggressive crawlers

**Current Implementation Audit Checklist:**
```
[ ] Verify sitemap URL is correct: https://www.boardblocs.com/sitemap.xml
[ ] Confirm Googlebot has proper access
[ ] Confirm Bingbot has proper access
[ ] Verify no important pages are blocked
[ ] Confirm backup/test files are excluded
[ ] Test with Google Search Console robots.txt tester
```

### 3.2 sitemap.xml Requirements

**Location:** /sitemap.xml

**Required Elements:**
- [ ] Valid XML 1.0 with UTF-8 encoding
- [ ] Proper namespace declaration
- [ ] All 46 active pages included
- [ ] Correct canonical URLs for each page
- [ ] Appropriate priority values (0.0-1.0)
- [ ] Accurate lastmod dates
- [ ] Appropriate changefreq values

**Priority Guidelines:**
| Page Type | Priority | Changefreq |
|-----------|----------|------------|
| Homepage | 1.0 | weekly |
| Core Conversion Pages | 0.9 | monthly |
| Feature Pages | 0.8 | monthly |
| Industry Verticals | 0.7 | monthly |
| Blog/Resources | 0.6 | weekly |
| Role Pages | 0.6 | monthly |
| Legal Pages | 0.3 | yearly |

**Audit Checklist:**
```
[ ] Validate XML structure with W3C validator
[ ] Verify all 46 pages are listed
[ ] Check for 404 URLs in sitemap
[ ] Confirm URLs match canonical tags
[ ] Submit to Google Search Console
[ ] Submit to Bing Webmaster Tools
[ ] Verify lastmod dates are current
```

### 3.3 Canonical URL Requirements

**Every page MUST have:**
```html
<link rel="canonical" href="https://www.boardblocs.com/[page-name].html">
```

**Audit Checklist per Page:**
```
[ ] Canonical tag present in <head>
[ ] URL matches actual page URL
[ ] Uses HTTPS protocol
[ ] Uses www subdomain consistently
[ ] No trailing slashes (except homepage)
[ ] No duplicate canonical tags
[ ] Self-referencing canonical is valid
```

### 3.4 URL Structure Requirements

**Required Format:**
- Protocol: `https://`
- Subdomain: `www.`
- Domain: `boardblocs.com`
- Path: `/[page-name].html`

**Audit Checklist:**
```
[ ] All internal links use absolute URLs or correct relative paths
[ ] No broken internal links (404 errors)
[ ] No redirect chains (multiple redirects)
[ ] URLs are lowercase
[ ] URLs use hyphens (not underscores)
[ ] No dynamic parameters in canonical URLs
[ ] HTTP redirects to HTTPS (301)
[ ] Non-www redirects to www (301)
```

### 3.5 HTTPS/SSL Requirements

```
[ ] Valid SSL certificate
[ ] Certificate not expired
[ ] All resources loaded over HTTPS
[ ] No mixed content warnings
[ ] HSTS header enabled (recommended)
```

### 3.6 Crawlability Requirements

```
[ ] No noindex tags on important pages
[ ] No nofollow on internal links (unless intentional)
[ ] JavaScript-rendered content is crawlable
[ ] CSS/JS files not blocked in robots.txt
[ ] Server response time < 200ms
[ ] No soft 404 errors
```

## 4. On-Page SEO Requirements

### 4.1 Title Tag Requirements

**Specifications:**
- Length: 50-70 characters (optimal)
- Maximum: 60 characters for full display
- Format: `[Primary Keyword] - [Secondary Keyword] | BoardBlocs`

**Required Elements:**
- [ ] Unique title for each page
- [ ] Primary keyword near beginning
- [ ] Brand name included
- [ ] Compelling for CTR
- [ ] No keyword stuffing
- [ ] No duplicate titles across site

**Audit Checklist per Page:**
```
[ ] Title tag present
[ ] Length within 50-70 characters
[ ] Primary keyword included
[ ] Brand name included
[ ] No duplicate of another page
[ ] Matches page content
[ ] Action-oriented where appropriate
```

### 4.2 Meta Description Requirements

**Specifications:**
- Length: 150-160 characters (optimal)
- Maximum: 155 characters for full display

**Required Elements:**
- [ ] Unique description for each page
- [ ] Primary keyword included naturally
- [ ] Clear value proposition
- [ ] Call-to-action where appropriate
- [ ] Compelling for CTR
- [ ] Accurate representation of page content

**Audit Checklist per Page:**
```
[ ] Meta description present
[ ] Length within 150-160 characters
[ ] Primary keyword included
[ ] Compelling copy for clicks
[ ] No duplicate descriptions
[ ] Matches page content
[ ] Includes CTA where appropriate
```

### 4.3 Meta Keywords Requirements

**Note:** While meta keywords have minimal direct SEO value, they serve as documentation and may have minor benefit.

**Specifications:**
- Count: 5-15 keywords per page
- Format: Comma-separated, lowercase

**Audit Checklist:**
```
[ ] Keywords tag present
[ ] 5-15 relevant keywords
[ ] No keyword stuffing
[ ] Keywords match page content
[ ] Mix of short and long-tail keywords
```

### 4.4 Heading Structure Requirements

**H1 Tag:**
- [ ] Exactly ONE H1 per page
- [ ] Contains primary keyword
- [ ] Accurately describes page content
- [ ] Different from title tag (but related)

**Heading Hierarchy:**
- [ ] Logical progression (H1 → H2 → H3)
- [ ] No skipped levels
- [ ] Keywords in H2s where natural
- [ ] Descriptive and scannable

**Audit Checklist per Page:**
```
[ ] Single H1 tag present
[ ] H1 contains primary keyword
[ ] Proper H2-H6 hierarchy
[ ] No heading level skips
[ ] Headings describe content sections
[ ] No empty heading tags
```

### 4.5 Image SEO Requirements

**Alt Text:**
- [ ] All images have alt attributes
- [ ] Alt text is descriptive
- [ ] Keywords included naturally
- [ ] 125 characters or less
- [ ] Empty alt="" for decorative images

**Image Optimization:**
- [ ] Descriptive file names (hyphens, not underscores)
- [ ] Compressed for web (< 200KB typical)
- [ ] Appropriate dimensions
- [ ] WebP format with fallbacks (recommended)
- [ ] Lazy loading for below-fold images

**Audit Checklist:**
```
[ ] All content images have alt text
[ ] Alt text is descriptive and relevant
[ ] File names are SEO-friendly
[ ] Images are appropriately sized
[ ] No missing image files (broken images)
[ ] Title attributes where helpful
```

### 4.6 Internal Linking Requirements

**Requirements:**
- [ ] All pages accessible within 3 clicks from homepage
- [ ] Descriptive anchor text (not "click here")
- [ ] No broken internal links
- [ ] Related pages linked contextually
- [ ] Footer/navigation links consistent
- [ ] No orphaned pages

**Audit Checklist:**
```
[ ] Homepage links to all major sections
[ ] Related content is cross-linked
[ ] Anchor text is keyword-rich but natural
[ ] No nofollow on internal links (unless intentional)
[ ] Breadcrumbs implemented (recommended)
[ ] No circular redirects
```

### 4.7 Robots Meta Tag Requirements

**Standard Implementation:**
```html
<meta name="robots" content="index, follow, max-image-preview:large, max-snippet:-1, max-video-preview:-1">
<meta name="googlebot" content="index, follow">
```

**Audit Checklist:**
```
[ ] Robots meta tag present on all indexable pages
[ ] Correct index/follow directives
[ ] No accidental noindex tags
[ ] Googlebot-specific directives if needed
[ ] Snippet/image preview settings optimized
```

**Recommended on:** Pages with FAQ sections

```json

      }
    }
  ]

```
